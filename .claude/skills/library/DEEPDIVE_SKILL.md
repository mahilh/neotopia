## PURPOSE
Maximum depth analysis of any content, decision, or system.
10 mandatory steps. Cannot skip. LLM Council activated.
Every flaw found gets an exact fix. Self-improves after each use.

## 10-STEP SEQUENCE (mandatory · no skipping)
STEP 1: CONTEXT LOAD
Read everything provided. Search for anything unknown.
Flag time-sensitive items first (deadlines · security · blockers).

STEP 2: EMERGENCY SCAN
Scan for: security vulnerabilities · exposed secrets · legal risks
Flag these FIRST before any analysis.

STEP 3: XRAY! (brutal audit)
Rate /200. Score every dimension separately.
Name specific line numbers and elements causing deductions.

STEP 4: LLM COUNCIL (5 advisors)
BRUTUS: "Does this drive the outcome?"
SOPHIA: "Is the design premium or mediocre?"
MARCUS: "What are the technical risks?"
ISABELLA: "Would a sophisticated user trust this?"
CAESAR: "What is irreversible here? Highest-stakes risk?"
Each advisor: one finding + one fix. Pick the most critical.

STEP 5: RESEARCH LAYER
Web search to verify every claim and number.
Find newer/better alternatives. Surface what was missed.

STEP 6: FLAW INVENTORY (exhaustive)
CRITICAL: blocks success / legal risk / security risk
SIGNIFICANT: reduces quality / creates confusion
MINOR: polish / optimization
Number every flaw. No limit on count.

STEP 7: AUTO-FIX GENERATION
Every CRITICAL: exact fix (code / copy / action / command)
Every SIGNIFICANT: fix or decision needed
Every MINOR: batched copy-paste solutions

STEP 8: PATH TO 200/200
Separate into: code fixes · content fixes · dependencies · structural
Assign to: T1 / T2 / T3 / Mahil / external

STEP 9: SELF-IMPROVEMENT LOOP
What new rule was discovered?
What pattern worked well?
What should be added to this skill?
Write to .claude/comms/tomorrow.md

STEP 10: SAVE REPORT
Date · content analyzed · score · all flaws · all fixes · ONE THING (most impactful improvement)
Sync to Drive: node scripts/sync-drive-skills.cjs --skill deepdive
