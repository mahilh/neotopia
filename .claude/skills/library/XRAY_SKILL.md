# XRAY SKILL v2.0 · Rating: 190/200 · Trigger: XRAY!
# Brutal audit /200 · evidence mandate on every finding

## PURPOSE
Deep audit of any output, code, design, or decision.
Every finding must have specific evidence. No vague criticisms.
Rates /200. Identifies path to 200/200 with exact steps.

## SEQUENCE
1. Read everything provided (code, screenshot, output, doc)
2. State what's being audited and the rating scale
3. Score each dimension separately with specific deductions
4. Name exact line numbers / elements / commits causing each deduction
5. Separate: CRITICAL (blocks success) · SIGNIFICANT (reduces quality) · MINOR (polish)
6. For every CRITICAL: generate the exact fix
7. Show path to 200/200: what each missing point requires

## SCORING DIMENSIONS (200 points total)
Technical correctness: 0-50
Evidence quality: 0-40
Rule compliance (69 rules): 0-30
Cross-lane discipline: 0-30
Commit quality: 0-25
Session discipline: 0-25

## RULES
Every finding needs: WHAT · WHERE · WHY IT MATTERS · HOW TO FIX
Never say "could be improved" without saying exactly how
Never deduct points without citing specific evidence
Never give 200/200 — there is always something to improve
