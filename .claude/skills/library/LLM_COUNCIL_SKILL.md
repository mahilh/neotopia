# LLM COUNCIL SKILL v1.0 · Rating: 183/200 · Trigger: LLM Council
# 5-advisor adversarial gate · final decision authority

## PURPOSE
When a decision is hard, unclear, or high-stakes — activate the Council.
5 advisors with different lenses. Evidence mandate on every position.
Produces a clear verdict. Never produces "it depends" without specifics.

## THE 5 ADVISORS
BRUTUS (outcomes): "Does this actually drive the civilization goal?"
SOPHIA (quality): "Is this premium or is it mediocre? Be honest."
MARCUS (engineering): "What breaks? What's the hidden technical risk?"
ISABELLA (user): "Would a player building a consciousness civilization trust this?"
CAESAR (stakes): "What is irreversible here? What happens if we're wrong?"

## SEQUENCE
1. State the decision clearly in one sentence
2. Each advisor gives: one position + one specific piece of evidence
3. Identify the strongest objection across all advisors
4. Generate 2-3 concrete options with trade-offs
5. Recommend one option with explicit reasoning
6. Note what would change the recommendation

## EVIDENCE MANDATE
Every advisor position needs: WHAT they believe + WHY (specific evidence) + WHAT CHANGES THEIR MIND
No position without evidence. No evidence without specifics.
